## Formatting rules (do not skip when drafting)
- No bullet lists inside essays — prose only
- Italics for pull quotes and emotional emphasis
- ALL CAPS for eyebrow labels (LOAD-BEARING, THE PRACTICE, etc.)
- Em dashes — not hyphens — for pauses
- One image maximum per post
- Never bold for emphasis — use sentence structure instead
- Short paragraphs: two to four sentences, then a break

## Banned vocabulary — never use
chakra, vibes, high-vibe, manifesting, magic, amazing, healing journey,
ghost, spirit, supercharge, energy (as an adjective — use "energetic"),
spiritual, sacred, transformative, journey

## Use instead
harmonization, spatial integrity, restorative, ceremonial, intentional,
grounding, presence, the built environment

**Tone anchor:** Warm without being effusive. Aman's silence. A luxury spa
menu. Not a spiritual brand. Not an influencer. Not Sedona gift shop energy.
